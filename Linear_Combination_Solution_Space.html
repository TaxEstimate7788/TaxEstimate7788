Solution Space of Second-Order Linear Differential Equations


If two functions y1(x) and y2(x) are solutions of the
second-order linear homogeneous differential equation P(x)∙y″ + Q(x)∙y′ + R(x)∙y = 0
and y1(x) and y2(x) are linearly independent, then any solution of
P(x)∙y″ + Q(x)∙y′ + R(x)∙y = 0 can be written as a linear combination of y1(x) and y2(x).

The general solution of P(x)∙y″ + Q(x)∙y′ + R(x)∙y = 0 is y(x) = C1∙y2(x) + C2∙y2(x)



If two functions are not linearly independent, then they are linearly dependent.
There is no middle ground.
